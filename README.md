# 🧮 Epic Scientific Calculator

An interactive, terminal-based calculator designed for efficiency and simplicity. Built with C, this project demonstrates basic logic, loops, and user input handling.

## ✨ Key Features
* **Core Math Operations:** Addition, Subtraction, Multiplication, and Division.
* **Error Handling:** Built-in protection against division by zero.
* **Formatted Output:** Results are displayed with two-decimal precision.
* **User-Friendly Interface:** Clear, interactive menu system.
